1. AWS EC2 생성하기
2. EC2 접속(Putty 또는 인스턴스 연결)
3. 접속후 다음과 같은 순서로 변경  
	$ sudo passwd root  
	$ sudo vi  /etc/ssh/sshd_config  
    (38 : PermitRootLogin no를 yes로 변경)  
    (65 : PasswordAuthentication no를 yes로 변경)  

	$ sudo mkdir /root/.ssh  
	$ sudo cp /home/ec2-user/.ssh/authorized_keys /root/.ssh  
	$ sudo service sshd restart
